## 1. معالجة النصوص Text Processing

### 1.1 تنظيف النص Text Cleaning

تنظيف النص يتضمن إزالة الأحرف الخاصة، الأرقام، والمسافات الزائدة، وتوحيد حالة الأحرف.

```python
import re
import string

def clean_text(text):
    """تنظيف النص"""
    # إزالة HTML tags
    text = re.sub(r'<[^>]+>', '', text)
    # تحويل إلى أحرف صغيرة
    text = text.lower()
    # إزالة علامات الترقيم
    text = text.translate(str.maketrans('', '', string.punctuation))
    # إزالة المسافات المتعددة
    text = re.sub(r'\s+', ' ', text).strip()
    return text
```

### 1.2 Text Normalization توحيد النصوص

| الطريقة | الوصف | مثال |
|---------|-------|------|
| Lowercase | تحويل لكل الأحرف الصغيرة | Hello → hello |
| Stopwords removal | إزالة الكلمات الشائعة | the, a, is |
| Stemming | استخراج الجذر | studying → studi |
| Lemmatization | تحويل للمصدر الصحيح | studying → study |

### 1.3 Common Pitfalls | المشاكل الشائعة

```
⚠️ Danger: استخدام stemming فقط دون lemmatization
   ✗: running → runni (خطأ)
   ✓: running → run (صحيح)

⚠️ Danger: عدم التعامل مع Unicode
   ✗: استخدام ASCII فقط
   ✓: استخدام str.normalize()
```

## 2. التقسيم إلى وحدات Text Tokenization

### 2.1 Word Tokenization تقسيم الكلمات

Tokenization هي عملية تقسيم النص إلى وحدات (tokens) منفصلة للمعالجة.

```python
# ============================================
# تقسيم الجملة إلى كلمات
# ============================================

def word_tokenize(text):
    """تقسيم النص إلى كلمات"""
    # التعامل مع علامات الترقيم
    text = re.sub(r'([.,!?;])', r' \1 ', text)
    words = text.split()
    return [w for w in words if w.strip()]

# مثال
text = "المرحبا, كيف حالك؟ أنا بخير."
print(word_tokenize(text))
# ['مرحباً', ',', 'كيف', 'حالك', '?', 'أنا', 'بخير', '.']
```

### 2.2 Sentence Tokenization تقسيم الجمل

```python
import re

def sentence_tokenize(text):
    """تقسيم النص إلى جمل"""
    # تحديد نهاية الجملة
    sentence_endings = r'[.!?]+[?\s]+'
    sentences = re.split(sentence_endings, text)
    return [s.strip() for s in sentences if s.strip()]
```

### 3.2 Hidden Markov Model (HMM)

نموذج ماركوف المخفي يستخدم الاحتمالات transitions و emissions.

```python
# ============================================
# HMM POS Tagger - مثال مبسط
# ============================================

class HMMTagger:
    def __init__(self):
        # احتمالات الانتقال: P(tag_i | tag_{i-1})
        self.transition_probs = {
            ('DET', 'NOUN'): 0.8,
            ('NOUN', 'VERB'): 0.7,
            ('VERB', 'DET'): 0.6,
        }
        # احتمالات الإصدار: P(word | tag)
        self.emission_probs = {
            ('الكتاب', 'NOUN'): 0.9,
            ('يقرأ', 'VERB'): 0.8,
            ('ال', 'DET'): 0.95,
        }
    
    def tag(self, sentence):
        """وسم الجملة"""
        words = sentence.split()
        tags = []
        prev_tag = 'START'
        
        for word in words:
            best_tag = 'NOUN'  # افتراضي
            # اختيار أفضل tag
            for tag in ['NOUN', 'VERB', 'ADJ']:
                trans = self.transition_probs.get((prev_tag, tag), 0.1)
                emit = self.emission_probs.get((word, tag), 0.01)
                score = trans * emit
                if score > self.transition_probs.get((prev_tag, best_tag), 0) * \
                   self.emission_probs.get((word, best_tag), 0.01):
                    best_tag = tag
            tags.append(best_tag)
            prev_tag = best_tag
        
        return list(zip(words, tags))
```

## 4. التحليل اللغوي Parsing

### 4.1 constituency Parsing تحليل التراكيب

تحليل بنية الجملة إلى Constituents (VP, NP, etc.)

```python
# ============================================
# Context-Free Grammar
# ============================================

grammar = {
    'S': ['NP VP'],
    'NP': ['DET N', 'N', 'ADJ N'],
    'VP': ['V', 'V NP', 'V ADJ'],
    'DET': ['ال', 'هذا', 'ذلك'],
    'N': ['كتاب', 'طالب', 'مدرسة'],
    'V': ['يقرأ', 'يذهب', 'يكتب'],
    'ADJ': ['كبير', 'صغير', 'جديد'],
}

def parse_cky(sentence, grammar):
    """CYK Parsing Algorithm"""
    words = sentence.split()
    n = len(words)
    
    # Initialize chart
    chart = [[set() for _ in range(n)] for __ in range(n)]
    
    # Base case:单个词语
    for i, word in enumerate(words):
        for cat, prods in grammar.items():
            if word in prods:
                chart[i][i].add(cat)
    
    # Fill chart
    for length in range(2, n + 1):
        for start in range(n - length + 1):
            end = start + length
            for mid in range(start + 1, end):
                for left in chart[start][mid - 1]:
                    for right in chart[mid][end - 1]:
                        for cat, prods in grammar.items():
                            if f"{left} {right}" in prods:
                                chart[start][end - 1].add(cat)
    
    return 'S' in chart[0][n - 1]
```

### 5.2 Self-Attention الميول الذاتي

$$Attention(Q, K, V) = softmax\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

```python
import numpy as np
import math

def scaled_dot_product_attention(Q, K, V):
    """حساب attention مع scale"""
    d_k = Q.shape[-1]
    
    # حساب الدرجات
    scores = np.matmul(Q, K.T) / math.sqrt(d_k)
    
    # softmax
    attention_weights = softmax(scores, axis=-1)
    
    # تطبيق على القيم
    output = np.matmul(attention_weights, V)
    
    return output, attention_weights

def softmax(x, axis=-1):
    """دالة softmax"""
    exp_x = np.exp(x - np.max(x, axis=axis, keepdims=True))
    return exp_x / np.sum(exp_x, axis=axis, keepdims=True)
```

### 5.3 Multi-Head Attention رؤوس متعددة

```python
class MultiHeadAttention:
    def __init__(self, d_model, num_heads):
        self.d_model = d_model
        self.num_heads = num_heads
        self.d_k = d_model // num_heads
        
        # projections
        self.W_Q = [np.random.randn(d_model, self.d_k) for _ in range(num_heads)]
        self.W_K = [np.random.randn(d_model, self.d_k) for _ in range(num_heads)]
        self.W_V = [np.random.randn(d_model, self.d_k) for _ in range(num_heads)]
        self.W_O = np.random.randn(d_model, d_model)
    
    def forward(self, Q, K, V):
        outputs = []
        
        for i in range(self.num_heads):
            # projection
            Q_i = Q @ self.W_Q[i]
            K_i = K @ self.W_K[i]
            V_i = V @ self.W_V[i]
            
            # attention
            out_i, _ = scaled_dot_product_attention(Q_i, K_i, V_i)
            outputs.append(out_i)
        
        # دمج كل الرؤوس
        concatenated = np.concatenate(outputs, axis=-1)
        
        # projection نهائي
        output = concatenated @ self.W_O
        
        return output
```

## 6. تحليل المشاعر Sentiment Analysis

### 6.1 الطرق الكلاسيكية Classical Methods

```python
# ============================================
# تحليل المشاعر - طريقة القاموس
# ============================================

# قاموس الكلمات الإيجابية والسلبية
positive_words = {
    'ممتاز': 3, 'رائع': 3, 'جيد': 2, 'مثالي': 3,
    'ممتاز': 3, 'سعيد': 2, 'حب': 3, 'أفضل': 3,
    'great': 3, 'excellent': 3, 'good': 2, 'love': 3
}

negative_words = {
    'سيء': -3, 'فظيع': -3, 'كره': -3, 'أسوأ': -3,
    'سيء': -2, 'حزين': -2, 'فظيع': -3,
    'bad': -3, 'terrible': -3, 'hate': -3, 'awful': -3
}

def sentiment_score(text):
    """حساب درجة المشاعر"""
    words = text.lower().split()
    score = 0
    
    for word in words:
        if word in positive_words:
            score += positive_words[word]
        elif word in negative_words:
            score += negative_words[word]
    
    if score > 0:
        return 'positive', score
    elif score < 0:
        return 'negative', score
    else:
        return 'neutral', score
```

### 6.3 نماذج التدريب Training Models

```python
import torch
import torch.nn as nn
from transformers import BertModel

class SentimentClassifier(nn.Module):
    def __init__(self, num_classes=3):
        super().__init__()
        self.bert = BertModel.from_pretrained('bert-base-arabert')
        self.dropout = nn.Dropout(0.1)
        self.classifier = nn.Linear(768, num_classes)
    
    def forward(self, input_ids, attention_mask):
        outputs = self.bert(
            input_ids=input_ids,
            attention_mask=attention_mask
        )
        
        # استخدام [CLS]token
        pooled_output = outputs.pooler_output
        pooled_output = self.dropout(pooled_output)
        logits = self.classifier(pooled_output)
        
        return logits
```
