## 1. تخطيط الإنتاج Production Planning

### 1.1 مقدمة Introduction

تخطيط الإنتاج هو عملية تحديد ماذا، كم، ومتى يجب أن ننتج لتلبية الطلب مع تقليل التكاليف.

$$Total\_Cost = C_{production} + C_{inventory} + C_{shortage}$$

```python
# ============================================
# Production Planning Model
# ============================================

import numpy as np
from scipy.optimize import linprog

def production_planning(demand, production_cost, holding_cost, shortage_cost):
    """
    تخطيط الإنتاج الأمثل
    
    Variables:
    - x[t]: وحدات منتجة في periode t
    - i[t]: مخزون في نهاية periode t
    - s[t]: نقص في periode t
    
    Constraints:
    - i[t] = i[t-1] + x[t] - demand[t] + s[t]
    """
    n_periods = len(demand)
    
    # دالة الهدف: تقليل التكاليف
    c = np.concatenate([
        production_cost,      # تكلفة الإنتاج
        holding_cost,       # تكلفة التخزين
        shortage_cost      # تكلفة النقص
    ])
    
    # قيود: معادلة التوازن
    A_eq = []
    b_eq = []
    
    for t in range(n_periods):
        row = np.zeros(3 * n_periods)
        
        # x[t]
        row[t] = 1
        
        # i[t] (if t > 0)
        if t > 0:
            row[n_periods + t - 1] = 1
        
        # i[t]
        row[n_periods + t] = -1
        
        # s[t]
        if t > 0:
            row[2 * n_periods + t - 1] = 1
        
        row[2 * n_periods + t] = 1
        
        A_eq.append(row)
        b_eq.append(demand[t])
    
    A_eq = np.array(A_eq)
    b_eq = np.array(b_eq)
    
    # قيود: غير سالبة
    bounds = [(0, None) for _ in range(3 * n_periods)]
    
    # حل
    result = linprog(c, A_eq=A_eq, b_eq=b_eq, bounds=bounds)
    
    return result.x[:n_periods], result.x[n_periods:2*n_periods]
```

## 2. التحكم في المخزون Inventory Control

### 2.1 مقدمة Introduction

المخزون هو تخزين المواد والمنتجات لإشباع الطلب. الهدف هو تحديد كمية الطلب الأمثل(Order Quantity) ووقت الطلب (Reorder Point).

### 2.2 Economic Order Quantity (EOQ)

$$EOQ = \sqrt{\frac{2DS}{H}}$$

حيث:
- $D$: الطلب السنوي (Annual Demand)
- $S$: تكلفة إصدار الطلب (Ordering Cost)
- $H$: تكلفة الاحتفاظ (Holding Cost)

```python
# ============================================
# EOQ Model
# ============================================

import math

def eoq(annual_demand, ordering_cost, holding_cost):
    """حساب EOQ"""
    eoq = math.sqrt((2 * annual_demand * ordering_cost) / holding_cost)
    return eoq

def reorder_point(annual_demand, lead_time_days):
    """حساب نقطة إعادة الطلب"""
    daily_demand = annual_demand / 365
    return daily_demand * lead_time_days

def total_cost_annual(annual_demand, ordering_cost, holding_cost, unit_cost, qty):
    """التكلفة السنوية الإجمالية"""
    ordering_cost_annual = (annual_demand / qty) * ordering_cost
    holding_cost_annual = (qty / 2) * holding_cost
    purchasing_cost = annual_demand * unit_cost
    
    return ordering_cost_annual + holding_cost_annual + purchasing_cost

# مثال
D = 10000  # units/year
S = 50     # $ per order
H = 2      # $ per unit per year

optimal_qty = eoq(D, S, H)
print(f"EOQ: {optimal_qty:.0f} units")

# التحقق
min_cost = total_cost_annual(D, S, H, 10, optimal_qty)
print(f"Total Cost: ${min_cost:.2f}")
```

```python
# ============================================
# EOQ with Quantity Discounts
# ============================================

def eoq_with_discounts(annual_demand, ordering_cost, holding_cost_rate, price_breaks):
    """
    EOQ مع خصومات الكمية
    
    price_breaks: [(quantity, unit_price), ...]
    """
    # ترتيب حسب الكمية
    breaks = sorted(price_breaks, key=lambda x: x[0])
    
    best_qty = None
    best_cost = float('inf')
    
    for i, (break_qty, unit_price) in enumerate(breaks):
        H = holding_cost_rate * unit_price
        
        # حساب EOQ مع هذا السعر
        h = break_qty / 2  # مخزون متوسط
        eoq = math.sqrt(2 * annual_demand * ordering_cost / H)
        
        # تقريب للـ break quantity إذا لزم
        if eoq < break_qty:
            qty = break_qty
        else:
            qty = eoq
        
        # حساب التكلفة
        ordering = (annual_demand / qty) * ordering_cost
        holding = (qty / 2) * H
        purchasing = annual_demand * unit_price
        
        total = ordering + holding + purchasing
        
        if total < best_cost:
            best_cost = total
            best_qty = qty
    
    return best_qty, best_cost
```

### 2.4 Safety Stock المخزون الاحتياطي

$$SS = z \times \sigma_{demand} \times \sqrt{LT}$$

أو:

$$SS = (D_{max} \times LT) - (D_{avg} \times LT)$$

```python
# ============================================
# Safety Stock Calculation
# ============================================

import math
from scipy import stats

def safety_stock_normal(daily_demand_mean, daily_demand_std, lead_time, service_level=0.95):
    """حساب المخزون الاحتياطي (توزيع عادي)"""
    z = stats.norm.ppf(service_level)
    ss = z * daily_demand_std * math.sqrt(lead_time)
    return ss

def safety_stock_max(daily_demand_mean, lead_time, max_daily_demand):
    """حساب المخزون الاحتياطي (أقصى طلب)"""
    avg_demand = daily_demand_mean * lead_time
    max_demand = max_demand * lead_time
    return max_demand - avg_demand
```

### 2.5 ABC Analysis تحليل ABC

تحليل ABC يقسم المخزون إلى ثلاث فئات حسب القيمة:

| الفئة | النسبة من العناصر | النسبة من القيمة |_policy |
|--------|-------------------|------------------|----------|
| A | 20% | 80% |严格的 inventory control |
| B | 30% | 15% | moderate control |
| C | 50% | 5% | simplistic control |

```python
# ============================================
# ABC Analysis
# ============================================

def abc_analysis(inventory_data):
    """
    تحليل ABC للمخزون
    
    inventory_data: [(item_id, annual_value), ...]
    """
    # حساب القيمة الإجمالية
    total_value = sum(value for _, value in inventory_data)
    
    # حساب النسبة التراكمية
    items = sorted(inventory_data, key=lambda x: x[1], reverse=True)
    
    categorized = {'A': [], 'B': [], 'C': []}
    cumulative = 0
    
    for item_id, value in items:
        cumulative += value
        percentage = (cumulative / total_value) * 100
        
        if percentage <= 80:
            categorized['A'].append(item_id)
        elif percentage <= 95:
            categorized['B'].append(item_id)
        else:
            categorized['C'].append(item_id)
    
    return categorized
```

# ============================================
# MRP Implementation
# ============================================

class MRP:
    def __init__(self, bom, inventory, lead_times):
        """
        bom: {parent: [(child, quantity), ...]}
        inventory: {item: quantity_on_hand}
        lead_times: {item: lead_time_days}
        """
        self.bom = bom
        self.inventory = inventory.copy()
        self.lead_times = lead_times
    
    def explode_requirements(self, item, needed_qty):
        """تفجير المتطلبات"""
        if item not in self.bom:
            return [(item, needed_qty)]
        
        requirements = []
        
        for child, child_qty in self.bom[item]:
            total_child_qty = needed_qty * child_qty
            requirements.append((child, total_child_qty))
        
        return requirements
    
    def calculate_mrp(self, master_schedule):
        """
        حساب MRP
        
        master_schedule: [(period, item, qty), ...]
        """
        # متطلبات gross
        gross_reqs = {}
        for period, item, qty in master_schedule:
            if item not in gross_reqs:
                gross_reqs[item] = {}
            gross_reqs[item][period] = qty
        
        # متطلبات net
        net_reqs = {}
        
        for item, periods in gross_reqs.items():
            on_hand = self.inventory.get(item, 0)
            net_reqs[item] = {}
            
            for period, gross in sorted(periods.items()):
                net = max(0, gross - on_hand)
                net_reqs[item][period] = net
                
                on_hand = max(0, on_hand - gross)
                on_hand += net  # إضافة بعد الطلب
        
        # planned orders
        planned_orders = {}
        
        for item, periods in net_reqs.items():
            lead_time = self.lead_times.get(item, 0)
            planned_orders[item] = {}
            
            for period, net in periods.items():
                if net > 0:
                    order_period = period - lead_time
                    if order_period > 0:
                        planned_orders[item][order_period] = net
        
        return {
            'gross': gross_reqs,
            'net': net_reqs,
            'planned_orders': planned_orders
        }
```

class KanbanCard:
    def __init__(self, item_id, container_capacity):
        self.item_id = item_id
        self.container_capacity = container_capacity
    
    def signal_reorder(self, currentInventory):
        """إشارة إعادة الطلب"""
        if currentInventory <= self.container_capacity:
            return True
        return False

class KanbanSystem:
    def __init__(self):
        self.kanbans = {}
    
    def add_kanban(self, item_id, container_capacity):
        """إضافة kanban"""
        self.kanbans[item_id] = KanbanCard(item_id, container_capacity)
    
    def check_reorder(self, item_id, current_inventory):
        """فحص إذا يلزم إعادة الطلب"""
        if item_id in self.kanbans:
            return self.kanbans[item_id].signal_reorder(current_inventory)
        return False
    
    def calculate_containers(self, demand_rate, lead_time, container_capacity):
        """حساب عدد kanbans"""
        # عدد الحاويات = (طلب × وقت التسليم) / سعة الحاوية + مخزون احتياطي
        num_containers = (demand_rate * lead_time) / container_capacity + 1
        return int(math.ceil(num_containers))

# ============================================
# Single Machine Scheduling
# ============================================

import itertools

def single_machine_scheduling(jobs, method='SPT'):
    """
    جدولة آلة واحدة
    
    jobs: [(job_id, processing_time, weight, due_date), ...]
    method: SPT, EDD, WSPT, CR
    """
    if method == 'SPT':  # Shortest Processing Time
        jobs = sorted(jobs, key=lambda x: x[1])
    elif method == 'EDD':  # Earliest Due Date
        jobs = sorted(jobs, key=lambda x: x[2])
    elif method == 'WSPT':  # Weighted SPT
        jobs = sorted(jobs, key=lambda x: x[0] / x[1])
    elif method == 'CR':  # Critical Ratio
        # CR = (due_date - current_time) / processing_time
        jobs = sorted(jobs, key=lambda x: (x[2] - 0) / x[1])
    
    schedule = []
    time = 0
    total_tardiness = 0
    
    for job_id, p, w, due in jobs:
        schedule.append((job_id, time, time + p))
        tardiness = max(0, (time + p) - due)
        total_tardiness += tardiness
        time += p
    
    return schedule, total_tardiness
```

```python
# ============================================
# Johnson's Algorithm for Two-Machine Flow Shop
# ============================================

def johnsons_rule(jobs):
    """
   .job: (id, time_machine1, time_machine2)
    
    Rule:
    - إذا t1 ≤ t2: وضع في البداية
    - إذا t1 > t2: وضع في النهاية
    """
    # تقسيم Jobs
    group1 = []  # t1 ��� t2
    group2 = []  # t1 > t2
    
    for job in jobs:
        t1, t2 = job[1], job[2]
        if t1 <= t2:
            group1.append(job)
        else:
            group2.append(job)
    
    # ترتيب
    group1.sort(key=lambda x: x[1])  # ascending t1
    group2.sort(key=lambda x: x[2], reverse=True)  # descending t2
    
    # دمج
    schedule = group1 + group2
    
    # حساب makespan
    m1_finish = 0
    m2_finish = 0
    
    for job in schedule:
        m1_finish += job[1]
        m2_finish = max(m2_finish, m1_finish) + job[2]
    
    return schedule, m2_finish
```

## 6. Capacity Planning تخطيط الطاقة

### 6.1Rough-Cut Capacity Planning

```python
# ============================================
# Rough-Cut Capacity Planning
# ============================================

def rough_cut_capacity_planning(mps, routing, capacity_standards):
    """
    تخطيط طاقةROUGH-CUT
    
    mps: [(period, product, qty), ...]
    routing: {product: [(work_center, hours), ...]}
    capacity_standards: {work_center: capacity_hours}
    """
    # حساب ساعات العمل لكل فترة
    required_capacity = {}
    
    for period, product, qty in mps:
        if product not in routing:
            continue
        
        for work_center, hours in routing[product]:
            if work_center not in required_capacity:
                required_capacity[work_center] = {}
            
            if period not in required_capacity[work_center]:
                required_capacity[work_center][period] = 0
            
            required_capacity[work_center][period] += qty * hours
    
    # مقارنة بالطاقة المتاحة
    analysis = {}
    
    for work_center, period_hours in required_capacity.items():
        available = capacity_standards.get(work_center, 0)
        
        analysis[work_center] = {}
        
        for period, required in period_hours.items():
            utilization = (required / available) * 100 if available > 0 else 0
            status = 'OK' if utilization <= 100 else 'OVER'
            
            analysis[work_center][period] = {
                'required': required,
                'available': available,
                'utilization': utilization,
                'status': status
            }
    
    return analysis
```
