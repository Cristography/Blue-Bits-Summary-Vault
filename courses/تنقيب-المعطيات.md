## 1. قواعد الارتباط Association Rules

### 1.1 مقدمة Introduction

قواعد الارتباط تربط بين العناصر المتكررة معاً في قاعدة البيانات. مثال كلاسيكي: customers الذين يشترون diapers يشترون أيضاً beer.

$$X \Rightarrow Y$$

حيث:
- $X$: antecedent (السبب)
- $Y$: consequent (النتيجة)

### 1.2 مقاييس الدعم Measures

| المقياس | الصيغة | الوصف |
|----------|--------|-------|
| Support | $supp(X) = \frac{|X|}{|D|}$ | نسبة تحتوي على X |
| Confidence | $conf(X\Rightarrow Y) = \frac{supp(X\cup Y)}{supp(X)}$ | نسبة X تحتوي على Y |
| Lift | $lift = \frac{supp(X\cup Y)}{supp(X) \times supp(Y)}$ | التحسن على العشوائي |

```python
# ============================================
# حسابAssociation Rules
# ============================================

from itertools import combinations

def calculate_support(itemset, transactions):
    """حساب support ل itemset"""
    count = sum(1 for t in transactions if itemset.issubset(t))
    return count / len(transactions)

def calculate_confidence(rule, transactions):
    """حساب confidence"""
    antecedent, consequent = rule
    union = antecedent | consequent
    supp_union = calculate_support(union, transactions)
    supp_ant = calculate_support(antecedent, transactions)
    return supp_union / supp_ant if supp_ant > 0 else 0

def calculate_lift(rule, transactions):
    """حساب lift"""
    antecedent, consequent = rule
    union = antecedent | consequent
    supp_union = calculate_support(union, transactions)
    supp_ant = calculate_support(antecedent, transactions)
    supp_cons = calculate_support(consequent, transactions)
    
    expected = supp_ant * supp_cons
    return supp_union / expected if expected > 0 else 0

# مثال
transactions = [
    {'milk', 'bread', 'eggs'},
    {'milk', 'bread'},
    {'milk', 'eggs'},
    {'bread', 'eggs'},
    {'milk', 'bread', 'eggs', 'butter'},
]

items = {'milk', 'bread', 'eggs', 'butter'}

# حساب كل القواعد الممكنة
rules = []
for a in items:
    for b in items - {a}:
        rule = ({a}, {b})
        support = calculate_support({a, b}, transactions)
        confidence = calculate_confidence(rule, transactions)
        lift = calculate_lift(rule, transactions)
        rules.append((rule, support, confidence, lift))

# ترتيب حسب lift
rules.sort(key=lambda x: x[3], reverse=True)
for rule, supp, conf, lift in rules[:5]:
    print(f"{rule[0]} → {rule[1]}: supp={supp:.2f}, conf={conf:.2f}, lift={lift:.2f}")
```

### 1.3 Apriori Algorithm

```
┌─────────────────────────────────────────────────────────────────┐
│                    Apriori Algorithm                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Input: min_support, transactions                                 │
│  Output: Frequent itemsets                                     │
│                                                                 │
│  1. L₁ = {frequent 1-itemsets}                               │
│  2. k = 2                                                     │
│  3. while Lₖ₋₁ ≠ ∅:                                           │
│  4.   Cₖ = generate_candidates(Lₖ₋₁)                        │
│  5.   for transaction in transactions:                      │
│  6.     for candidate in Cₖ:                                 │
│  7.       if candidate ⊆ transaction:                        │
│  8.         candidate.count++                                │
│  9.   Lₖ = {c ∈ Cₖ | c.count ≥ min_support}                 │
│  10.  k++                                                     │
│  11. return ∪ₖ Lₖ                                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```python
# ============================================
# Apriori Algorithm Implementation
# ============================================

def apriori(transactions, min_support=0.5):
    """تطبيق خوارزمية Apriori"""
    n = len(transactions)
    
    # جميع العناصر الفريدة
    items = set()
    for t in transactions:
        items.update(t)
    
    # L₁: الـ items المتكررة
    frequent = {}
    for item in items:
        count = sum(1 for t in transactions if item in t)
        support = count / n
        if support >= min_support:
            frequent[frozenset([item])] = count
    
    # التكرارات اللاحقة
    all_frequent = [frequent]
    k = 2
    
    while all_frequent[k-2]:
        # توليد المرشحين
        prev_items = list(all_frequent[k-2].keys())
        candidates = generate_candidates(prev_items, k)
        
        # حساب الدعم
        candidate_counts = {c: 0 for c in candidates}
        for t in transactions:
            t_frozen = frozenset(t)
            for c in candidates:
                if c.issubset(t_frozen):
                    candidate_counts[c] += 1
        
        # تصفية حسب min_support
        current = {c: count for c, count in candidate_counts.items() 
                   if count/n >= min_support}
        all_frequent.append(current)
        k += 1
    
    return all_frequent

def generate_candidates(prev_itemsets, k):
    """توليد مرشحين من itemsets السابقة"""
    candidates = set()
    
    for i, itemset1 in enumerate(prev_itemsets):
        for itemset2 in prev_itemsets[i+1:]:
            union = itemset1 | itemset2
            if len(union) == k:
                # فحص كل الـ (k-1)-subsets
                valid = True
                for item in union:
                    subset = union - {item}
                    if subset not in prev_itemsets:
                        valid = False
                        break
                if valid:
                    candidates.add(union)
    
    return candidates
```

### 2.2 K-Means Algorithm

```python
# ============================================
# K-Means Implementation
# ============================================

import numpy as np
import random

def kmeans(data, k, max_iterations=100, tolerance=1e-4):
    """خوارزمية K-Means"""
    
    # اختيار مراكز عشوائية
    centroids = random.sample(list(data), k)
    centroids = np.array(centroids)
    
    for iteration in range(max_iterations):
        # تعيين كل نقطة لأقرب مركز
        clusters = [[] for _ in range(k)]
        
        for point in data:
            distances = [np.linalg.norm(point - c) for c in centroids]
            nearest = np.argmin(distances)
            clusters[nearest].append(point)
        
        # حساب المراكز الجديدة
        new_centroids = []
        for cluster in clusters:
            if cluster:
                new_centroid = np.mean(cluster, axis=0)
            else:
                new_centroid = centroids[len(new_centroids)]
            new_centroids.append(new_centroid)
        
        new_centroids = np.array(new_centroids)
        
        # فحص التقارب
        if np.linalg.norm(new_centroids - centroids) < tolerance:
            centroids = new_centroids
            break
        
        centroids = new_centroids
    
    return centroids, clusters
```

### 2.3 DBSCAN Algorithm

```python
# ============================================
# DBSCAN Implementation
# ============================================

def dbscan(data, epsilon, min_pts):
    """خوارزمية DBSCAN"""
    n = len(data)
    visited = [False] * n
    clusters = []
    
    def get_neighbors(point_idx):
        """جلب الجيران ضمن epsilon"""
        neighbors = []
        point = data[point_idx]
        
        for i in range(n):
            if i != point_idx:
                distance = np.linalg.norm(point - data[i])
                if distance <= epsilon:
                    neighbors.append(i)
        
        return neighbors
    
    def expand_cluster(point_idx, cluster):
        """توسيع الـ cluster"""
        visited[point_idx] = True
        cluster.append(point_idx)
        
        neighbors = get_neighbors(point_idx)
        
        for neighbor_idx in neighbors:
            if not visited[neighbor_idx]:
                visited[neighbor_idx] = True
                
                # فحص إذا كان core point
                if len(get_neighbors(neighbor_idx)) >= min_pts:
                    expand_cluster(neighbor_idx, cluster)
            elif neighbor_idx not in cluster:
                cluster.append(neighbor_idx)
    
    # معالجة كل النقاط
    for i in range(n):
        if not visited[i]:
            neighbors = get_neighbors(i)
            
            if len(neighbors) >= min_pts:
                # بدء cluster جديد
                cluster = []
                expand_cluster(i, cluster)
                clusters.append(cluster)
    
    return clusters
```

### 2.4 Evaluation Metrics معايير التقييم

| المقياس | الصيغة | الوصف |
|----------|--------|-------|
| Inertia | $\sum d(c_i, x)^2$ | مجموع المسافات لمركز الـ cluster |
| Silhouette | $\frac{b-a}{max(a,b)}$ | جودة التجميع |
| Davies-Bouldin | $\frac{1}{k}\sum R_{ij}$ | التشابه بين الـ clusters |
| Rand Index | $\frac{TP+TN}{TP+TN+FP+FN}$ | مقارنة بـ labels الحقيقية |

```python
# ============================================
# Silhouette Score Calculation
# ============================================

def silhouette_score(data, labels):
    """حساب silhouette score"""
    n = len(data)
    k = len(set(labels))
    
    scores = []
    
    for i in range(n):
        cluster_i = labels[i]
        
        # a(i): متوسط المسافة لنقاط نفس الـ cluster
        same_cluster = [j for j in range(n) if labels[j] == cluster_i and j != i]
        if same_cluster:
            a_i = np.mean([np.linalg.norm(data[i] - data[j]) for j in same_cluster])
        else:
            a_i = 0
        
        # b(i): أقل متوسط لمسافة لـclusters الأخرى
        min_b = float('inf')
        for c in range(k):
            if c != cluster_i:
                other_cluster = [j for j in range(n) if labels[j] == c]
                if other_cluster:
                    b_c = np.mean([np.linalg.norm(data[i] - data[j]) 
                                   for j in other_cluster])
                    min_b = min(min_b, b_c)
        
        if min_b != float('inf') and max(a_i, min_b) > 0:
            s_i = (min_b - a_i) / max(a_i, min_b)
            scores.append(s_i)
    
    return np.mean(scores)
```

### 3.2 Decision Tree شجرة القرار

```python
# ============================================
# Decision Tree Classifier
# ============================================

import numpy as np
from collections import Counter

class DecisionTreeClassifier:
    def __init__(self, max_depth=None, min_samples_split=2):
        self.max_depth = max_depth
        self.min_samples_split = min_samples_split
        self.tree = None
    
    def _gini_impurity(self, y):
        """حساب gini impurity"""
        if len(y) == 0:
            return 0
        
        counts = Counter(y)
        probs = [count / len(y) for count in counts.values()]
        return 1 - sum(p**2 for p in probs)
    
    def _best_split(self, X, y):
        """البحث عن أفضل split"""
        best_gain = -1
        best_feature = None
        best_threshold = None
        
        n_features = X.shape[1]
        base_gini = self._gini_impurity(y)
        
        for feature in range(n_features):
            thresholds = set(X[:, feature])
            
            for threshold in thresholds:
                left_mask = X[:, feature] <= threshold
                right_mask = ~left_mask
                
                if sum(left_mask) == 0 or sum(right_mask) == 0:
                    continue
                
                y_left = y[left_mask]
                y_right = y[right_mask]
                
                n = len(y)
                gain = base_gini - (len(y_left)/n * self._gini_impurity(y_left) +
                                   len(y_right)/n * self._gini_impurity(y_right))
                
                if gain > best_gain:
                    best_gain = gain
                    best_feature = feature
                    best_threshold = threshold
        
        return best_feature, best_threshold
    
    def _build_tree(self, X, y, depth=0):
        """بناء الشجرة"""
        n_samples = len(y)
        n_labels = len(set(y))
        
        # stopping criteria
        if (n_samples < self.min_samples_split or 
            (self.max_depth and depth >= self.max_depth) or
            n_labels == 1):
            return Counter(y).most_common(1)[0][0]
        
        feature, threshold = self._best_split(X, y)
        
        if feature is None:
            return Counter(y).most_common(1)[0][0]
        
        left_mask = X[:, feature] <= threshold
        right_mask = ~left_mask
        
        left_tree = self._build_tree(X[left_mask], y[left_mask], depth+1)
        right_tree = self._build_tree(X[right_mask], y[right_mask], depth+1)
        
        return {
            'feature': feature,
            'threshold': threshold,
            'left': left_tree,
            'right': right_tree
        }
    
    def fit(self, X, y):
        """تدريب النموذج"""
        self.tree = self._build_tree(np.array(X), np.array(y))
    
    def predict(self, X):
        """تنبؤ"""
        return np.array([self._predict_single(x, self.tree) for x in X])
    
    def _predict_single(self, x, node):
        """تنبؤ لنقطة واحدة"""
        if isinstance(node, dict):
            if x[node['feature']] <= node['threshold']:
                return self._predict_single(x, node['left'])
            else:
                return self._predict_single(x, node['right'])
        else:
            return node
```

### 3.4 Evaluation Metrics

```python
# ============================================
# Classification Evaluation Metrics
# ============================================

def confusion_matrix(y_true, y_pred):
    """إنشاء مصفوفة الالتباس"""
    classes = sorted(set(y_true))
    n = len(classes)
    
    matrix = {c1: {c2: 0 for c2 in classes} for c1 in classes}
    
    for true, pred in zip(y_true, y_pred):
        matrix[true][pred] += 1
    
    return matrix

def accuracy(y_true, y_pred):
    """الدقة"""
    correct = sum(1 for t, p in zip(y_true, y_pred) if t == p)
    return correct / len(y_true)

def precision(y_true, y_pred, class_label):
    """الدقة لفئة معينة"""
    true_positive = sum(1 for t, p in zip(y_true, y_pred) 
                     if t == class_label and p == class_label)
    predicted_positive = sum(1 for p in y_pred if p == class_label)
    
    return true_positive / predicted_positive if predicted_positive > 0 else 0

def recall(y_true, y_pred, class_label):
    """الاستدعاء لفئة معينة"""
    true_positive = sum(1 for t, p in zip(y_true, y_pred) 
                     if t == class_label and p == class_label)
    actual_positive = sum(1 for t in y_true if t == class_label)
    
    return true_positive / actual_positive if actual_positive > 0 else 0

def f1_score(y_true, y_pred, class_label):
    """F1-Score"""
    prec = precision(y_true, y_pred, class_label)
    rec = recall(y_true, y_pred, class_label)
    
    return 2 * prec * rec / (prec + rec) if (prec + rec) > 0 else 0
```

### 4.1 مقدمة Introduction

شجرة القرار هي هيكل شجري حيث كل node يمثل اختبار على feature، و كل edge يمثل نتيجة الاختبار، و كل leaf يمثل 클래스.

### 4.2 تقسيم المعلومات Information Gain

$$Entropy(S) = -\sum_{c} p_c \log_2(p_c)$$

$$IG(S, A) = Entropy(S) - \sum_{v} \frac{|S_v|}{|S|} Entropy(S_v)$$

```python
# ============================================
# Information Gain Calculation
# ============================================

import math
from collections import Counter

def entropy(labels):
    """حساب الانتروبيا"""
    counts = Counter(labels)
    probs = [count / len(labels) for count in counts.values()]
    return -sum(p * math.log2(p) for p in probs if p > 0)

def information_gain(data, labels, feature_idx, threshold):
    """حساب information gain"""
    # تقسيم البيانات
    left_mask = data[:, feature_idx] <= threshold
    right_mask = ~left_mask
    
    if sum(left_mask) == 0 or sum(right_mask) == 0:
        return 0
    
    # الانتروبيا الأصلية
    H_S = entropy(labels)
    
    # الانتروبيا بعد التقسيم
    n = len(labels)
    H_left = entropy(labels[left_mask])
    H_right = entropy(labels[right_mask])
    
    H_after = (sum(left_mask)/n * H_left + sum(right_mask)/n * H_right)
    
    return H_S - H_after
```

### 4.3 Gini Index

$$Gini(S) = 1 - \sum_{c} p_c^2$$

```python
def gini_index(labels):
    """حساب Gini Index"""
    counts = Counter(labels)
    probs = [count / len(labels) for count in counts.values()]
    return 1 - sum(p**2 for p in probs)
```

### 5.4 مثال عملي

```java
// Weka API Example
import weka.core.Instances;
import weka.core.converters.ConverterUtils.DataSource;
import weka.classifiers.trees.J48;
import weka.classifiers.Evaluation;

public class WekaExample {
    public static void main(String[] args) throws Exception {
        // تحميل البيانات
        DataSource source = new DataSource("data.arff");
        Instances data = source.getDataSet();
        data.setClassIndex(data.numAttributes() - 1);
        
        // إنشاء classifier
        J48 tree = new J48();
        tree.setUnpruned(true);
        
        // تدريب
        tree.buildClassifier(data);
        
        // تقييم
        Evaluation eval = new Evaluation(data);
        eval.crossValidateModel(tree, data, 10, new java.util.Random(1));
        
        System.out.println(eval.toSummaryString());
        System.out.println(eval.toMatrixString());
    }
}
```
